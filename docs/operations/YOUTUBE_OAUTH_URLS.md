# YouTube OAuth URLs

Canonical setup page:

- `https://caios.vercel.app/integrations/youtube`

Canonical OAuth endpoints:

- `/api/integrations/youtube/connect`
- `/api/integrations/youtube/callback`

Compatibility redirects are retained for previously shared URLs:

- `/api/youtube/auth`
- `/api/youtube/callback`

Google Cloud must continue using the canonical callback URI:

- `https://caios.vercel.app/api/integrations/youtube/callback`
