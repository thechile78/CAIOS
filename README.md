# CAIOS — Chilemaniacs Newsroom OS

CAIOS is the internal operating system for Chilemaniacs.com.

## Current Milestone

**v4.0 — AI Editorial Command Center**

The goal of v4.0 is to combine story discovery, verification, clustering, drafting, approvals, WordPress draft creation, analytics, alerts, and newsroom activity into one production-grade application.

## Core Rule

**AI prepares. The Chile approves. Nothing publishes automatically.**

## Planned Stack

- Next.js
- TypeScript
- Supabase
- WordPress
- OpenAI
- Google Search Console
- Google Analytics 4
- Vercel

## v4.0 Modules

- Command Center dashboard
- Story Intelligence Engine
- Editorial Workflow Engine
- AI Writing Engine
- Publishing Engine
- Analytics Intelligence
- Source and alert monitoring
- Asset and image-rights tracking
- Editorial calendar
- Audit and activity log

## Development Workflow

1. Work on feature branches.
2. Open a pull request into `main`.
3. Run lint, typecheck, tests, and production build.
4. Review all publishing and approval controls.
5. Merge only after verification.

See `AGENTS.md` and `docs/V4_IMPLEMENTATION_PLAN.md` for project rules and the execution roadmap.
